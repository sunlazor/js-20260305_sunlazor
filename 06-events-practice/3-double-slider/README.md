# DoubleSlider компонент 

Необходимо реализовать "DoubleSlider" компонент, который понадобится на странице 
[products](https://course-js.javascript.ru/products)

!["DoubleSlider"](double-slider.gif)

**Обратите внимание:** 

"DoubleSlider" компонент должен иметь возможность "диспатчнуть" событие
"range-select", подписавшись на которое, таблица продуктов сможет загрузить
новые данные с сервера.

Подробнее про генерация пользовательских событий смотри в [учебнике](https://learn.javascript.ru/dispatch-events)

## Требования к реализации

Работа ведется в файле `src/index.ts`.

### 1. Интерфейс класса

```typescript
new DoubleSlider({ min, max, formatValue, selected })
```

* `min`: нижняя граница диапазона.
* `max`: верхняя граница диапазона.
* `formatValue`: функция форматирования значений.
* `selected`: начальный диапазон `{ from, to }`.

### 2. Публичные свойства

* `element`: корневой DOM-элемент слайдера.
* `min`: минимальное значение диапазона.
* `max`: максимальное значение диапазона.

### 3. Методы

* `destroy()`: удаляет компонент и очищает обработчики.

### 4. Поведение

* При перемещении ползунков обновляются значения диапазона.
* Событие `range-select` генерируется при завершении изменения диапазона, `detail` содержит `{ from, to }`.
* Корректно обрабатывается случай, когда `min` и `max` совпадают.

## Инструкция по запуску

**Запуск в режиме разработки:**
```bash
npm run dev:06:3-double-slider
```

**Запуск тестов:**
```bash
npm test 06-events-practice/3-double-slider
```
