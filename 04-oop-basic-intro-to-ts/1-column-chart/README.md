# ColumnChart

Необходимо реализовать компонент «ColumnChart» цель которого показывать статистику продаж в графическом виде на главной странице [проекта](https://course-js.javascript.ru/)

![ColumnChart](column-chart.png)

## Описание задачи

Ваша цель — реализовать класс на TypeScript, который будет генерировать HTML-разметку графика, управлять его состоянием и предоставлять API для обновления данных без перезагрузки всей страницы.

Компонент должен:
1.  Принимать данные и настройки через конструктор.
2.  Отображать "скелетон" (заглушку загрузки), если данные отсутствуют.
3.  Корректно масштабировать столбцы графика относительно фиксированной высоты.
4.  Обновлять отображение при получении новых данных.

## Требования к реализации

Работа ведется в файле `src/index.ts`. Готовая верстка компонента находится в файле [static.html](static.html).
Класс должен обладать следующей структурой и функциональностью:

### 1. Интерфейс класса

```typescript
new ColumnChart(options)
```

Аргумент `options` — это объект с параметрами:
*   `data`: массив чисел (данные графика).
*   `label`: заголовок (например, "orders").
*   `value`: основное числовое значение (например, "344").
*   `link`: ссылка "View all" (если передана).
*   `formatHeading`: функция для форматирования значения `value` (например, добавление знака валюты `$`).

Если `data` пуст или не передан, компонент должен отображаться в состоянии загрузки (добавить CSS-класс `column-chart_loading` к корневому элементу).

### 2. Методы класса

*   `update(data)`: Принимает новый массив данных и обновляет только тело графика (столбцы), не перерисовывая весь компонент целиком.
*   `remove()`: Удаляет элемент компонента из DOM.
*   `destroy()`: Полностью удаляет компонент, очищает обработчики событий и ссылки на DOM-элементы (для предотвращения утечек памяти).

### 3. Свойства

*   `element`: Ссылка на корневой DOM-элемент компонента (`HTMLElement`).
*   `chartHeight`: Фиксированная высота графика (должна быть равна `50`).

## Подсказка

### Расчет высоты столбцов

Высота графика жестко задана в CSS переменной `--chart-height` и свойстве класса `chartHeight` (равна 50px). Данные могут быть любыми, поэтому их нужно пропорционально масштабировать.

1.  Найдите максимальное значение в данных (`maxValue`).
2.  Вычислите коэффициент масштабирования: `scale = 50 / maxValue`.
3.  Для каждого элемента данных:
    *   Значение для CSS (в пикселях): `Math.floor(item * scale)`. Это значение идет в `style="--value: ..."`.
    *   Значение для тултипа (в процентах): `(item / maxValue * 100).toFixed(0) + '%'`.

### Data-аттрибут
Обратите внимание на атрибуты `data-element`. Они полезны для быстрого доступа к внутренним элементам (заголовок, тело графика) при обновлении, чтобы не искать их каждый раз через `querySelector`.
